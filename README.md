# Промтехнопарк Митино — фактический финансовый дашборд

Готовый статический HTML-проект для публикации на GitHub Pages.

## Состав

- `index.html` — самодостаточный интерактивный дашборд с embedded JSON-данными.
- `assets/abcentrum-logo.png` — логотип ABCENTRUM для верхней панели.
- `assets/favicon-32.png`, `assets/favicon-64.png`, `assets/apple-touch-icon.png` — favicon/иконка вкладки на основе эмблемы логотипа.
- `assets/icons/*.svg` — встроенный subset Lucide Icons для UI-иконок.

## Запуск локально

Откройте `index.html` в браузере. Для графиков требуется доступ к CDN Apache ECharts.

## Публикация на GitHub Pages

1. Загрузите файлы проекта в репозиторий.
2. Включите GitHub Pages для ветки `main` / папки root.
3. Откройте опубликованную ссылку.

## Примечания

Расчет факта идет по сданным лотам. Вакантные площади показываются отдельно как площадь, доля вакантности и денежный потенциал по ставкам модели.

UI-иконки взяты из Lucide Icons; license notice сохранен в `assets/icons/NOTICE.md`.
